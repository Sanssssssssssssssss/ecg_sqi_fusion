# BUT v116 SQI Baseline Comparator

Independent feature-generation and legacy control experiment for classical SQI
baselines on the official BUT v116 split.

BUT is single-lead, while the SQI baseline expects the 84-feature 12-lead SQI
table. This experiment copies the BUT lead into pseudo-12-lead columns, then
recomputes QRS detections and SQIs from waveform.

This standalone runner keeps the older good-vs-rest control:

- `good -> 1`
- `medium/bad -> -1`

The current Chapter 4 BUT comparison is generated by
`src.supplemental_transformer_experiments.chapter4_evidence.run but-models`.
That table uses the corrected original-only three-class test scope with
SVM-RBF all84, SVM-RBF selected5, SQI LM-MLP OvR, and Conformer rows.

All outputs go to:

```powershell
outputs/transformer/supplemental/but_sqi_baseline/
```

Run:

```powershell
python -m src.supplemental_transformer_experiments.but_sqi_baseline.run pipeline --run
```

Dry-run:

```powershell
python -m src.supplemental_transformer_experiments.but_sqi_baseline.run pipeline
```
