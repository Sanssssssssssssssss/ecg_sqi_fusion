Rule family: bad if `pc1 > bad_pc1_min`; good if strict low-PC1/QRS-prom band passes; plus a wider high-QRS-prom good rescue band. Else medium.
A precision bad-stress veto is optionally applied as a transparent reportable rule.

## Interpretation
- A wider high-QRS-prom good rescue should recover the visually obvious good regressions caused by the narrow pc1 split.
- If original improves without breaking node gates, this is a better candidate for a simple transparent adapter than more synthetic-row training.
- If original good rises but medium falls too much, the adapter needs a counter-veto for visible-QRS medium detail rather than another broad training sweep.
