# ECG SQI Fusion

This project studies ECG signal-quality assessment using classical
signal-quality-index models and waveform Conformers. It covers public 12-lead
Set-A and single-lead BUT data and ships four frozen models for reproducible
inference.

## Start here

- [Quick Start](quickstart.md) — install, test, and run the main commands.
- [Inference and Docker](inference.md) — run any of the four packaged models.
- [Reproduction Guide](reproduction.md) — rebuild experiments in an isolated environment.
- [Report](report.md) — final report location and submission status.

## Repository scope

The repository contains the Python source, tests, Docker definitions,
reproduction controller, documentation, and four inference model artifacts.
Generated datasets, predictions, figures, and experiment outputs are rebuilt
under `outputs/` and are not committed.
